# Saint Demetrios Greek Orthodox Church

A static React SPA for a Greek Orthodox church website with a Byzantine/Orthodox visual identity featuring raccoon illustrations.

**Live site:** https://idvorkin-church.surge.sh

## Tech Stack

- React 19 + TypeScript + Vite 7 + Tailwind CSS v4
- `vite-plugin-singlefile` for single-file builds
- HashRouter for gisthost compatibility

## Commands

- `npm run dev` — Start dev server with HMR
- `npm run build` — Type-check and bundle
- `npm run lint` — ESLint
- `npm run preview` — Serve production build locally
